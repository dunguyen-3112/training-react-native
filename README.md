-   React: v.18.1.0
-   React Native: v.0.70.5
-   expo: v.47.0.12

-   Clone repository:

    > ```
    > git clone https://github.com/dunguyen-3112/training-react-native.git
    > ```

-   Check out the branch: feature/practice

    > ````
